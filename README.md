# KAI-Model Platform

The KAI-Model Platform is a web-based application designed to help users understand their AI readiness profile. It uses a comprehensive assessment to generate a personalized transformation plan based on cognitive psychology and task analysis.

## Features

- **AI Readiness Assessment:** A detailed questionnaire to evaluate a user's AI readiness.
- **Personalized Transformation Plan:** A 30-60-90 day plan generated by Google Gemini.
- **Cognitive Profile Analysis:** Determines whether a user is an Adaptor or an Innovator.
- **Organizational Dashboard:** A dashboard for organizations to manage and analyze team dynamics.

## Tech Stack

- **Frontend:** React, TypeScript, Vite
- **AI:** Google Gemini
- **Styling:** Tailwind CSS
- **UI Components:** Lucide React, Recharts

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
